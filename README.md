# Global-Literacy-Education-Trends
Project Overview

This project analyzes global literacy, illiteracy, GDP per capita, and schooling trends using Python, SQL, and Power BI.

The objective is to understand:

Global literacy improvements over time

Gender gaps in youth literacy

Relationship between GDP and education

Countries with high illiteracy despite schooling

Regional and global educational disparities

🛠 Tools & Technologies Used

Python (Pandas, NumPy)

SQLite (SQL Queries & Database Design)

Power BI (Data Visualization & Dashboarding)

Jupyter Notebook

GitHub

📊 Data Analysis Workflow
1️⃣ Data Cleaning & Processing (Python)

Removed duplicates

Standardized country & year formats

Calculated:

Illiteracy %

Literacy gender gap

GDP per schooling year

Growth rates

2️⃣ Database Design (SQLite)

Three structured tables were created:

literacy_rates

illiteracy_population

gdp_schooling

SQL queries were written to analyze:

Countries with illiteracy > 20%

GDP vs Schooling patterns

Gender literacy gap

Regional literacy averages

Ranking countries by GDP per schooling

3️⃣ Power BI Dashboard

The interactive dashboard includes:

✔ Global literacy trends
✔ Illiteracy heat map
✔ GDP vs Schooling scatter analysis
✔ Gender literacy gap comparison
✔ Country & year slicers
✔ KPI Cards

📈 Key Insights

Literacy rates have improved globally, but disparities remain.

Some countries show high schooling years but persistent illiteracy.

Strong correlation observed between GDP per capita and literacy.

Youth literacy gender gap is narrowing in many regions. Power BI.
